# Security Policy

## Supported Scope

This repository contains **educational material only** — Frida scripts and lessons for learning about SSL certificate pinning in a controlled lab environment.

## Reporting a Security Issue in This Repository

If you find a security problem with the scripts or documentation in this repository (e.g., a script that could cause unintended harm beyond its stated educational scope), please:

1. **Do not open a public GitHub Issue.**
2. Email the maintainer directly (see GitHub profile).
3. Describe the concern clearly.

We will respond within 7 days.

## Responsible Disclosure (General)

If during your authorized security research you discover a real vulnerability in Facebook's platform:

- Report it to Meta's Bug Bounty: https://hackerone.com/facebook
- Do **not** exploit or publicize it before Meta has patched it.
- See [lessons/05_ethics.md](lessons/05_ethics.md) for the full responsible disclosure process.

## Out of Scope

The following are **not** security issues in this repository:
- SSL pinning itself (it is an intentional protection, not a bug)
- The fact that these bypass techniques exist (they are publicly documented)
- Requests to add bypasses targeting specific user accounts (out of scope and illegal)
