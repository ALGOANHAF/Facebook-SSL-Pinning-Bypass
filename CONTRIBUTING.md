# Contributing to Facebook SSL Pinning Bypass Lab

Thank you for considering a contribution! This project is educational and community-driven.

## How to Contribute

### Report Issues
- Use GitHub Issues to report bugs or unclear explanations.
- Include your OS, Python version, Frida version, and Android version.

### Add Scripts
- Add new Frida scripts to the `scripts/` folder.
- Include a header comment explaining what the script does and which pinning layer it targets.
- Test your script before submitting.

### Improve Lessons
- Lessons are in `lessons/` as Markdown files.
- Keep language beginner-friendly.
- Include commands that can be copy-pasted directly.

### Pull Request Guidelines
1. Fork the repository.
2. Create a branch: `git checkout -b feature/your-feature`.
3. Write clear commit messages.
4. Open a PR with a description of what changed and why.

## Code of Conduct
- All contributions must be for **educational purposes only**.
- Do not add content that facilitates illegal activity.
- Be respectful in all discussions.
